GT7 Online Dealerships

Note because I know it's a bit confusing:
"X More Days Remaining" means "this day and X more" left in the dealership.
ie. "2 More Days Remaining" -> "1 More Day Remaining" -> "Last Day Remaining".
Hope that clears up any confusion.

If you want to make a bot that uses this data, a JSON file is built from the data and can be found at:
ddm999.github.io/gt7info/data.json
This will update when the site is updated.
Also, if you're wondering, the site is updated manually.

To better explain the problems with GT7's economy, estimated 'grind' and 'normal' playtimes have been added.
For 'grind', this assumes that you are extremely efficient and earn 900k Cr. per hour by doing nothing but the single current best event.
For 'normal', this assumes you're playing reasonably and earning 200k Cr. per hour but not spending a single credit until you can afford the car.
