GT7 Online Dealerships & Info

Note because I know it's a bit confusing:
"X More Days Remaining" means "this day and X more" left in the dealership.
ie. "2 More Days Remaining" -> "1 More Day Remaining" -> "Last Day Remaining".
Hope that clears up any confusion.

If you want to make a bot that uses this data, a JSON file is built from the data and can be found at:
ddm999.github.io/gt7info/data.json
This will update when the site is updated.
Also, if you're wondering, the site is updated manually.

All dates are (YY-MM-DD)
Recent changes:
- v1.11 is out! More UCD, LCD cars, less grinding, endurance events... (22-04-07)
- If a car is given as a reward, this is now noted in the dealerships. Hover over the icon to see where to get that car for free. (22-04-04)
- GT ENGINE now has a BoP listing for Gr.3! It's linked to in the Balance of Performance section. (22-03-30) (and now Gr.4! 22-03-31) (and now Gr.B / Gr.2! 22-04-05)
- Got rid of the "custom race" info in Legend Cars, it was mostly there as a joke and got in the way of the NEW label for the '70 Skyline (bc of it's really long name). (22-03-27)
- I've been made aware that a couple legendary cars were available for less time than estimated, so I've reduced the estimate. (22-03-23)
- Shuffled the sections around a bit to make the Daily Races closer to the top. (22-03-22)
- Daily Races have been added - note the styling is incomplete. (22-03-22) Eh, the styling isn't that bad. It's honestly just way too much effort to get it "correct". (22-04-05)

Breaking data.json changes:
- Booleans have all been changed to actual booleans instead of y/n strings. (22-03-24)
- For a short time, daily race specific cars were given as db IDs: that's been reverted back to names. (22-04-04)

Estimates assume cars will be available for 7 days after they appear.
For used cars it seems to actually be 7 to 9 days (so cars may appear for longer) - note that cars ARE always "Limited Stock" for 2 days, however.
